## Download Instructions

1. Go to https://www.flaticon.com/
2. Search for each service (e.g., "netflix logo")
3. Download as PNG (512px or 1024px recommended) or SVG
4. Rename to match the filename above
5. Save in this directory (`public/icons/streaming/`)
